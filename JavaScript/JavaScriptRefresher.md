### Destructuring
- Easily extract array elements or object properties and store them in variables

#### Array Destructuring

```
[a, b] = ["Hello", "Rolling"]
console.log(a) // Hello
console.log(b) // Rolling
```

#### Object Destructuring

```
{name} = {name: 'Yumi', age: 28}
console.log(name) // Yumi
console.log(age) // undefined
```
